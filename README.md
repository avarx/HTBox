# HTBox
[ansible-playbook-kali](https://github.com/iesplin/ansible-playbook-kali) customized for my needs.

How to use
------

- Install Kali
- Clone this repo 
    ```bash
    git clone https://github.com/avarx/HTBox
    ```
- Run the deploy script
    ```bash
    cd HTBox
    ./deploy.sh
    ```
